# Cobra Programing Language
A Very Simple Programing Language made in Python and Pygame that is made to be easy.

# Features
- Variables and math operations
- Control flow including If, Loop, Break, and input
- 2D graphics with, image rendering, rectangles, and text drawing
- Built-in button creation and click detection
- Sound playback using Pygame
- Debugging tools including variable watching and logging
